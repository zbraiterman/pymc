Analysis of An \(AR(1)\) Model in pyMC3¶

%matplotlib inline

Consider the following AR(1) process, initialized in the infinite past:

\[ \begin{align}\begin{aligned} y_t = \theta y_{t-1} + \epsilon_t,\\where :math:`\epsilon_t \sim iid{\cal N}(0,1)`. Suppose you’d like to\end{aligned}\end{align} \]

learn about \(\theta\) from a a sample of observations \(Y^T = \{ y_0, y_1,\ldots, y_T \}\).

First, let’s generate our sample.

In [2]:

np.random.seed(seed=1848)

T = 100
y = np.zeros((T,))

for i in range(1,T):
    y[i] = 0.95 * y[i-1] + np.random.normal()

plt.plot(y);

Consider the following prior for \(\theta\): \(\theta \sim {\cal N}(0,\tau^2)\). We can show that the posterior distribution of \(\theta\) is of the form

\[\theta |Y^T \sim {\cal N}( \tilde{\theta}_T, \tilde{V}_T),\]

where

\[\begin{split}\begin{eqnarray} \tilde{\theta}_T &=& \left( \sum_{t=1}^T y_{t-1}^2 + \tau^{-2} \right)^{-1} \sum_{t=1}^T y_{t}y_{t-1} \\ \tilde{V}_T &=& \left( \sum_{t=1}^T y_{t-1}^2 + \tau^{-2} \right)^{-1} \end{eqnarray}\end{split}\]
In [3]:

tau = 1.0
with pm.Model() as ar1:
    beta = pm.Normal('beta', mu=0, sigma=tau)
    data = pm.AR('y', beta, sigma=1.0, observed=y)
    trace = pm.sample(1000, cores=4)

pm.traceplot(trace);

Extension to AR(p)¶

We can instead estimate an AR(2) model using pyMC3.

\[ \begin{align}\begin{aligned} y_t = \phi_1 y_{t-1} + \phi_2 y_{t-2} + \epsilon_t.\\The ``AR`` distribution infers the order of the process by size the of\end{aligned}\end{align} \]

rho argmument passed to AR.

In [6]:

with pm.Model() as ar2:
    beta = pm.Normal('beta', mu=0, sigma=tau, shape=2)
    data = pm.AR('y', beta, sigma=1.0, observed=y)
    trace = pm.sample(1000, cores=4)

pm.traceplot(trace);

You can also pass the set of AR parameters as a list.

In [7]:

with pm.Model() as ar2:
    beta = pm.Normal('beta', mu=0, sigma=tau)
    beta2 = pm.Uniform('beta2')
    data = pm.AR('y', [beta, beta2], sigma=1.0, observed=y)
    trace = pm.sample(1000, tune=1000, cores=4)

pm.traceplot(trace);
